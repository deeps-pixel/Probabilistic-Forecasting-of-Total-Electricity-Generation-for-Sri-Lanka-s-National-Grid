# Probabilistic Forecasting of Total Electricity Generation for Sri Lanka's National Grid

This repository contains the code, models, and outputs for the project **"Probabilistic Forecasting of Total Electricity Generation for Sri Lanka's National Grid."**

This project was developed as part of an Individual Report for the module **IS 4007 - Statistics in Practice II**.

## Project Overview

The main objective of this project is to use machine learning techniques (specifically LightGBM) to forecast the electricity generation for Sri Lanka's national grid. The analysis involves probabilistic forecasting, error analysis, and feature importance evaluation using SHAP values.

## Repository Structure

- `00_master_forecast.ipynb`: The main Jupyter Notebook containing the data pipeline, model training (LightGBM), forecasting, and visualization code.
- `models/`: Contains the serialized trained machine learning models (`final_lightgbm.pkl`) and feature names (`feature_names.pkl`).
- `outputs/`: Stores the visualizations and figures generated by the notebook, including:
  - Actual vs. Predicted forecasts (`figure_actual_vs_predicted.png`)
  - Error distribution and analysis (`figure_error_by_type.png`, `figure_error_distribution.png`)
  - Feature importance analysis (`figure_shap_bar.png`, `figure_shap_summary.png`)
  - Sample forecasts (e.g., `figure_yugadhanavi_forecast.png`)
- `s16751_Report.pdf` & `00_master_forecast.pdf`: Project reports detailing the methodology, results, and conclusions.

## Note on Data

Large datasets (like `01_timeseries_data_imputed.csv`) used in this project have been excluded from the repository via `.gitignore` due to GitHub's file size limitations. 

## Requirements

To run the notebook, you will need Python 3 installed along with several data science libraries such as `pandas`, `numpy`, `matplotlib`, `seaborn`, `lightgbm`, `shap`, and `scikit-learn`.
